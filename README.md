# Career Guidance Chatbot

A machine learning-based web application that helps users choose suitable career paths based on their skills, interests, and academic background.

1.Live Demo

🔗 https://career-guidance-ddztsnrwtljxqbyi9tia4d.streamlit.app/

2. Tech Stack

   -Python
   -Machine Learning (KNN, Random Forest)
   -Streamlit

3.Features

  - Career prediction based on user input
  - Uses KNN for similarity-based recommendations
  - Uses Random Forest for accurate classification
  - Simple and interactive UI


#Project Summary

This project compares KNN and Random Forest algorithms to provide reliable career suggestions. Random Forest was used to improve prediction accuracy, while KNN helps in similarity-based recommendations.
